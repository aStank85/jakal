# Jakal V0.1 - Parser MVP

## Release Summary

- Version: `0.1.0`
- Focus: parser-first MVP
- Status: complete (historical milestone)

## Scope Delivered

- Copy-paste parser for R6 Tracker season drawer text
- Section-aware stat extraction (`game`, `rounds`, `combat`, `clutches`, `multikills`, `ranked`, `uncategorized`)
- Stat key normalization and numeric parsing (percentages, comma-separated values, time in hours)
- Critical-stat validation with explicit parse failures

## Files

- `src/parser.py`
- `src/__init__.py`
- `tests/test_parser.py`

## Out of Scope in v0.1

- Persistent storage
- Derived metrics and role scoring
- Player comparison workflows
- Terminal menu UI
- Insight generation

## Notes

This document captures the original parser MVP milestone only.  
Database, calculator, UI, and comparison functionality shipped in later versions.
