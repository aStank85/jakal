# Jakal v0.1.1 - Parser MVP

## Release Summary

- Version: `0.1.1`
- Focus: parser-first MVP
- Status: complete (historical milestone)

## Delivered

- Copy/paste parser for R6 Tracker season drawer text
- Section-aware extraction for game, rounds, combat, clutches, multikills, ranked, uncategorized
- Stat-key normalization and numeric parsing
- Critical-stat validation with explicit parse failures
- Terminal output formatting for quick review
